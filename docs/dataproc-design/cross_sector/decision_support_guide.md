# 🧠 Decision Support Guide

High-fidelity cross-sector linking and fuzzy matching for complex AI reasoning. This sector provides the architectural bridge between isolated industrial datasets.

## Status: **HARDENED**
- **Architecture**: Propagated Signal Linking / Entity Resolution.
- **Verification**: 100% Parity.

## Data Sources
- **Internal Correlator**: Proprietary cross-dataset linking engine.
- **Heuristics**: Signal propagation and fuzzy matching clusters.

## 🛠️ Schema (`StandardSchema`)
- `entity_a`: Source entity identifier.
- `entity_b`: Target entity identifier.
- `metric`: `linkage_confidence`, `signal_strength`, or `parity_score`.
- `value`: Numerical (0.0 to 1.0).
- `link_type`: `direct`, `fuzzy`, or `inferential`.
