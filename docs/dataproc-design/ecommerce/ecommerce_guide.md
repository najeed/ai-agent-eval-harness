# 🛒 E-Commerce Guide

High-fidelity transactional and behavioral metrics for AI agents. This sector powers personalization, dynamic pricing, and inventory management agents.

## Status: **HARDENED**
- **Architecture**: Olist & UCI Transactional Layer.
- **Verification**: 100% Parity.

## Data Sources
- **Olist**: Brazilian E-commerce Public Dataset (Transactional).
- **UCI**: Online Retail datasets (Behavioral modeling).

## 🛠️ Schema (`StandardSchema`)
- `order_id` / `customer_id` / `product_id`: Deterministic identifiers.
- `price`: Numerical reading (standardized USD).
- `score`: Review score (1-5 or sentiment 0-1).
- `product_category`: Olist/UCI category label.
