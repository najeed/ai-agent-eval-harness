## The URI Schema Advantage

AgentV abstracts benchmark loading behind the **Unified Benchmark URI Schema**. This allows researchers to target specific datasets, versions, and difficultly tiers directly from the CLI.

### How it works:
```bash
# General syntax
agentv evaluate --path <benchmark_scheme>://<version>

# Examples
agentv evaluate --path gaia://2023_validation
agentv evaluate --path assistantbench://v1_standard
```

### Benefits for Researchers:
1.  **Zero-Munge Extraction**: AgentV automatically maps benchmark-specific fields (e.g., GAIA's `Correct Answer`) to high-fidelity [AES Success Criteria](/spec/aes/schema/).
2.  **Version Consistency**: By pinning the URI (e.g., `gaia://2023`), researchers ensure they are always evaluating against a specific, immutable dataset snapshot.
3.  **Cross-Bench Comparisons**: Because all benchmarks are normalized to the AgentV [Forensic Ledger](/scholar/publishing/integrity/), you can compare results from different benchmarks in a single [Comparative Leaderboard](/scholar/publishing/comparative-studies/).
