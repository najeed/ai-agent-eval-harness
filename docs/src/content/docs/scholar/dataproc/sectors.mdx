## Generator Architecture

Each sector in the Dataproc Engine uses a dedicated **Generator** class. Researchers can extend these generators to create custom datasets for niche research areas.

### Deterministic Noise
Generators support **Synthetic Noise Injection**, allowing researchers to test agent robustness against:
- Data corruption (bit flips).
- Incomplete records (missing fields).
- Outdated information (temporal drift).

### Veracity Checks
Every generated record includes a `veracity_score` and a SHA-256 `source_hash`. This ensures that in a research setting, you can prove the lineage of every data point used in the evaluation.
