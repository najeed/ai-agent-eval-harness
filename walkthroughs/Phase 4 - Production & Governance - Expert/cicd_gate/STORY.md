# STORY: The Hard Gate

The deployment pipeline is a river of light, flowing from the developer's workstation to the global edge network. But at the final bridge, it stops.

"Why isn't the release moving?" the DevOps Lead asks, staring at the monitor. "All the unit tests passed. The build is green. The containers are signed."

"The tests passed," you say, pointing to a small, pulsing red icon labeled **Sentinel**. "But the evaluation failed. The agent's **Spearman's Rho** dropped to 0.6. Its reliability is no longer within the industrial safety margin."

"But it's only 0.1 off!"

"In this industry, 0.1 is the difference between an update and an outage," you reply. "The **Hard Gate** doesn't care about 'almost.' It only cares about 'verifiable.' Today, you'll learn how to set the Sentinel's threshold and watch as it automatically defends your production environment from a weak agent release."
