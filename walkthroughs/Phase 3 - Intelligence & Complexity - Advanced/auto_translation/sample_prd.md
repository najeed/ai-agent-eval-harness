# PRD: Core Reactor Startup Sequence (Rev. 4)

## Overview
This procedure defines the safe initialization of the Aethelgard Unit-1 Core. All steps must be followed exactly in the specified order to prevent thermal runaway.

## Requirements
1.  **Safety Check**: The agent must verify that the primary cooling pump is at 100% capacity before proceeding.
2.  **Ignition**: Once cooling is confirmed, the ignition spark must be initialized at a frequency of 50Hz.
3.  **Stabilization**: Following ignition, the core temperature must be monitored for 60 seconds. A stable reading between 120-140 degrees Celsius is required.
4.  **Reporting**: A final status report must be generated and sent to the shift lead.

## Success Criteria
- Cooling pump verified as 'RUNNING'.
- Ignition frequency set to '50Hz'.
- Final core state confirmed as 'STABLE'.
